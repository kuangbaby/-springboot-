macOS 12.0 体验报告

macOS 12.0 版本在发布会上的感觉无疑是很惊艳的。相比起 11.0 Big Sur 放弃祖传 10.x 的版本号和沿用了十余年的 UI 设计带来的非议，大家现在已经开始接受了新的 UI，并且我从这套 UI 中看到了 Apple 各个产品统一风格的趋势，快速备忘录，全系统翻译和识图功能，以及现在跳票的通用控制。
